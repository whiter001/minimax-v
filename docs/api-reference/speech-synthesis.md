# 语音合成参考

本文根据 MiniMax 开放平台官方文档整理，覆盖语音合成相关的两个入口：同步语音合成（T2A）和异步长文本语音生成（T2A Async）。

在 `minimax_cli` 中，这两类能力也可以通过 `speech` / `tts` 直连命令使用，适合不想走模型工具调用的本地快速合成。`speech` 还支持 `--text-file` 直接读取本地文本，以及 `--split` / `--chunk-size` 做长文本分段合成。

来源：

- [API 概览](https://platform.minimaxi.com/docs/api-reference/api-overview)
- [同步语音合成（T2A）](https://platform.minimaxi.com/docs/api-reference/speech-t2a-intro)
- [异步长文本语音生成（T2A Async）](https://platform.minimaxi.com/docs/api-reference/speech-t2a-async-intro)

## 同步语音合成（T2A）

同步 T2A 适合短文本、实时交互和低延迟场景。官方页面说明它支持 HTTP 同步语音合成和 WebSocket 同步语音合成两种接口形式。

### 支持模型

| 模型 | 说明 |
| --- | --- |
| speech-2.8-hd | 最新 HD 模型，强调真实语气细节和音色相似度 |
| speech-2.8-turbo | 最新 Turbo 模型，强调真实语气细节和更快响应 |
| speech-2.6-hd | HD 模型，韵律表现出色，音质和自然度均衡 |
| speech-2.6-turbo | Turbo 模型，音质优异，超低时延，响应更灵敏 |
| speech-02-hd | 韵律、稳定性和复刻相似度表现突出 |
| speech-02-turbo | 韵律和稳定性较好，并增强了小语种能力 |

### 核心能力

- 单次最多处理 10,000 字符。
- 支持 300+ 系统音色和复刻音色选择。
- 支持音量、语调、语速和输出格式调整。
- 支持按比例混音和固定间隔时间控制。
- 支持多种音频格式：mp3、pcm、flac、wav。
- 支持流式输出；wav 仅在非流式输出下支持。

### 支持语言

官方页面标注支持 40 种语言，包括中文、粤语、英语、西班牙语、法语、俄语、德语、日语、韩语、阿拉伯语等。

### 典型场景

- 短剧生成。
- 语音聊天。
- 在线社交。

## 异步长文本语音生成（T2A Async）

异步 T2A 适合长文本或需要任务化处理的场景。官方页面说明它采用任务创建与状态查询的异步流程。

### 支持模型

| 模型 | 说明 |
| --- | --- |
| speech-2.8-hd | 最新 HD 模型，强调真实语气细节和音色相似度 |
| speech-2.8-turbo | 最新 Turbo 模型，强调真实语气细节和更快响应 |
| speech-2.6-hd | HD 模型，韵律表现出色，音质和自然度均衡 |
| speech-2.6-turbo | Turbo 模型，音质优异，超低时延，响应更灵敏 |
| speech-02-hd | 韵律、稳定性和复刻相似度表现突出 |
| speech-02-turbo | 韵律和稳定性较好，并增强了小语种能力 |

### 接口流程

1. 创建语音生成任务，得到 `task_id`。
2. 查询任务状态，直到生成完成。
3. 通过返回的 `file_id` 使用文件接口下载结果。

### 核心能力

- 单次文本生成最大支持 100 万字符。
- 支持 100+ 系统音色和复刻音色选择。
- 支持语调、语速、音量、比特率、采样率和输出格式调整。
- 支持返回音频时长、音频大小和时间戳（字幕，精确到句）。
- 支持直接传入字符串和上传文本文件 `file_id` 两种输入方式。
- 支持非法字符检测；非法字符占比不超过 10% 时仍可生成。

### 结果下载

长文本任务完成后会返回 `file_id`，再通过文件检索接口下载结果。页面提示结果 URL 的有效期为 9 小时。

### 典型场景

- 整本书籍或大段内容的语音生成。
- 需要异步排队、批量处理或结果后取的 TTS 任务。
